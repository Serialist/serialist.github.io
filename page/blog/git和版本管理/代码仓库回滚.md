# 代码仓库回滚

## 步骤

1. 获取目标 commit 的 hash 值

2. 在命令行使用 `git reset --hard <commit hash>`

   e.g. `git reset --hard b14cc04`

## 怎么获取 commit 对应的 hash 值

在一个仓库中，他的每一个 commit 都对应一个唯一的 hash 值。

### 使用  vs code
