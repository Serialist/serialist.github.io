# 奇淫巧技 in C

## C99 复合字面量

```c
void foo(int data[4]);

void bar(){
	foo((int[4]){1000,
                 2000,
                 1000,
                 0});
}
```

1. 顾名思义，C99 标准的特性，不过大部分嵌入式都支持 C99 了吧
2. 复合，就是可以用于字符串、结构体等复合数据类型。

> [ C语言复合字面量应该如何使用？ - 知乎](https://zhuanlan.zhihu.com/p/385523590)

## C99 赋值结构体

```c
struct foo{
  int a;
  float b;
  char c;
};

void bar(){
    struct foo x = {
        .a = 114,
        .b = 51.4f,
        .c = 'f'
    };
}
```

1. 用`.xxx`的方式定义结构体内部
2. 指定初始化器
