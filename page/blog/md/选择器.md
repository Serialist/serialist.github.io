# 选择器

CSS 中使用选择器来指定 HTML 元素。

选择器包括以下五种：

- 基本选择器
- 属性选择器
- 伪类与伪元素选择器
- 关系选择器
- 选择器列表

## 基本选择器

**标签选择器（tag）**

即HTML中的标签

```css
span {
    color: red;
}
```

**类选择器（class）**

使用点号`.`和类名

```css
.myClass {
    color: blue;
}
```

**ID 选择器**

使用`#`号和ID

```css
#myID {
    color: green;
}
```
