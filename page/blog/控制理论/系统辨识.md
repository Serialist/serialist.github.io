# 系统辨识

为了深入研究一个系统，我们通常需要得到其传递函数。对于简单的系统，通过物理建模就可以得到。但是对于复杂的、分析困难的、类似黑箱的系统，就很通过建模难得传递函数了。

系统辨识就是：

- 分析系统的输入输出
- 寻找相近的系统模型
- 根据这个模型拟合传递函数

通过系统辨识，可以得到拟合的传递函数，以此进行控制理论上的分析。

## 系统辨识的要素

1. 采集数据

   采集目标系统的输入输出。

2. 确定模型集

   根据对系统的了解（先验知识）选出可能的模型，抛去完全不可能的模型。

   模型如：

   - 线性回归
   - 最小二乘估计
   - ARX 模型估计
   - 辅助变量法
   - 最小化准则

3. 参数辨识

   参数辨识就是：根据模型进行拟合，得到对应的参数

4. 模型验证
