# JS的基本数据类型和对象

在学习JS的过程中，有很多文章说JS中“一切数据皆对象”。但是如果这样，对象作为引用数据类型，又由什么组成呢？实际上，这个说法或许不够准确。

## 基本数据类型值不可变

JS中有着Number、String、BigInt、Boolean、Null、Undefined、NaN、Symbol几个基本数据类型

**基本数据类型是字面值**。实际上，JS的基本数据类型并不是对象。但我们在用一些基本数据类型赋值的时候，JS会值把隐形的**包装**在一个对象里，并提供一些便于操作的方法。或许可以说，JS创建的变量是对象，而对象中储存着作为字面值的基本类型数据。

在改变变量时，实际上只是把对象中的数据改成了另一个字面值。这和C语言中的指针和字符串的关系有些类似——我们不能改变字符串的值，但可以把指针指向另一个字符串。
