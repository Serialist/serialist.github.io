## 考古

虽然写的创建时间是 2024-10-05，但其实只有上面这两行是。这两行是整理电脑时，找到的 serialist.top 域名下的第一个 index.html。为了纪念就把这两行当作创建时间吧。

```html
<h1>Welcome to Serialist‘s!!!</h1>
<p>hello world</p>
```

在国庆节的时候，因为没什么事情做，于是学了一点点前端，整了个域名，白嫖了个服务器，于是又随便写了个 index 上去。后来服务器过期了就没再管。

第二年春天，有一天了解到 [MDUI](https://www.mdui.org/zh-cn/) 前端组件库，想起了胎死腹中的网站，故学习 MDUI，写了草率的页面放在 github page 上。然而由于水平不足，再加上不怎么会用 git，最终仓库变成了，于是把仓库删掉了。

暑假间又了解到了 Markdown 和 tex 公式，浅尝后发现前端渲染公式有些麻烦，于是最终用了 MKdocs 博客框架。用 Python 来构建网站确实很友好，总是不想去用 nodejs。在暑假期间也写了一些乱七八糟的前端知识、笔记等等。

后来 9 月左右，我了解到 [Hongliu](https://hongliu.icu) 也搭建了博客网站。此时我好久没动过我的网站，再打开顿时感觉风格不合意，决定捡起来重构。好在 AI 给力，加之又认真学了遍前端（之前学的不怎么认真hh），经过不断进化，最终改成了原生（除了 md 样式）的单页面驱动网站。其实一开始不了解 SPA，本来的想法是能够复用各种组件，如导航栏、页脚，内容都用 md 文档写，正好把 md 和 Tex 渲染也全交给前端好了，结果最后改着改着就成单页面了。

总之我也记不清了这个博客的历史，唯一找的到的是某些文件的创建时间，和 whois 上域名的注册记录。为了冒充老资历，就暂且当网站是 24 年 10 月创建的吧，但他真正活起来是在重构之后。

网站上有些文章之前发了，但重构忘记放回来了，有的文章是后来才发的，还有的移植时弄丢了（泣）。

## 关于博客

Blog，也就是 weB-LOG，用 web 作为自己的日志。

博客是互联网自留地，我其实很喜欢看博客，尤其喜欢看技术类和生活、旅游、摄影类的。在优秀博客里，我们可以看到一个人的比较真实的思考。

在这个媒体霸权被削弱、自媒体发展的时代，我一直认为博客却更像是一种传统媒体。博客是 “软单工通信” 的，没有社交应用式的 “围观”，而是一种自己和自己的对话。博客是给自己写的，有意营造人设则是推文。

我一直很向往自己写博客这件事，无论是技术类还是生活类。所以我在看到 Hongliu 也开始做博客时，也开始重新做了一些东西。我一直觉得，能够静下心来思考是难能可贵的事情，而我似乎常常难以静下心来做某件事。在大部分情况下，清醒的思考和冷静的行动的往往更有价值吧。
